This folder contains an implementation of the greedy algorithm to solve the integer traffic assignment problem (ITAP)
the file greedy_core.py contains the implementation of the greedy algorithm
the file example_greedy.ipynb contains an example of how to use the greedy algorithm

Requirements for greedy_core.py:
- Python 3.6 or higher
- Numpy
- Networkit
- copy

Additional Requirements for example_greedy.ipynb:
- matplotlib
- Networkx
